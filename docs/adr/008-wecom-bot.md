# ADR-008: 企微机器人集成架构

**日期**: 2026-07-11
**状态**: 已采纳
**决策者**: 曹子杰

## 背景

CHRONOS 的用户主要是团队管理者，日常工作在企微中完成。需要在企微内直接操作 CHRONOS（查看任务、搜索文件、切换项目），减少切换成本。

## 选项

1. **企微 H5 内嵌** — 开发简单但体验差
2. **企微机器人（消息回调）** — 纯文本命令，适合移动端
3. **企微小程序** — 体验最好但开发工作量大

## 决策

**选择企微机器人（自建应用消息回调）。**

关键设计：
- `POST /api/bot/callback` 接收加密 XML → 解密 → 解析命令 → 执行 → 加密回复
- 21 条自然语言命令（`/帮助`, `/任务`, `/搜索`, `/切换`, `/文件` 等）
- AES 加解密（`server/lib/bot/wecom-crypto.ts`）
- 文件分享码（4 位，24h 过期）
- 短链接（`/v/:short` → 文件预览页）
- news 卡片格式（上限 8 条）

## 后果

### 正面
- 企微内直接操作，零学习成本
- 加密通信符合企业安全要求
- 文件预览页支持图片/PDF/下载

### 负面
- 命令式交互不如 GUI 直观（21 条命令的记忆成本）
- news 卡片上限 8 条（企微限制）
- 消息推送有限频（每分钟 20 条）
